# redshoes 
Force Redirect your OS data to a socks5 proxy
\
\
\
Redshoes is a CLI application which help you control redsocks and rediredt your packets to redsocks using iptables.

# Build
for build use make \
`make` \
for build using gcc: \
`make CXX=g++` 

# Installation
dist files after build are these  \
`./redshoes ./bin/* `
